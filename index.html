JavaScript (JS)

A programming language common across all browsers

Operators

An operator performs some operation on single or multiple operands (data value) and produces a result.

Variables

A variable is a declared value assigned to a name. example myNum = 3

Variable naming conventions

It is recommended to use camelCase when naming variables

Function

A function takes an argument and parameters in order to complete a more complex task. Can also be a method.

Methods

a method is a function that is a called on something

Arguments

Arguments are the real values passed to and received by a function

Parameters

Parameters are the placeholder for a value in the function

Return

Return stops the function and kicks out an answer to a function.

Chaining methods

Calling multiple functions on the same object consecutively.

Strings

A variable that can hold multiple letters or numbers.

Booleans

True or False values.
